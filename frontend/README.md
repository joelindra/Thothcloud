# ThothCloud Frontend 🪐

This is the React + Vite frontend for the ThothCloud ecosystem.

## Tech Stack
- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** Tailwind CSS + custom glassmorphism tokens
- **Animations:** Framer Motion
- **Icons:** Lucide React

## Development
To start the frontend dev server:
```bash
npm install
npm run dev
```

For the full project documentation, please see the [Main README](../README.md).
